# La Maison Fleurie - Élégance et Fleurs en Harmonie

Een verfijnde oase van bloemen, thee en Franse elegantie. Ontdek onze Tea Room en Boutique in een harmonieuze, bloemrijke setting.

## Technologieën
- **React 19** met **TypeScript**
- **Vite** voor snelle ontwikkeling
- **Tailwind CSS** voor styling
- **shadcn/ui** voor elegante componenten
- **Firebase** (Firestore & Auth) voor de webshop en beheer
- **Motion** voor verfijnde animaties

## Aan de slag

### Vereisten
- Node.js (versie 18 of hoger)
- npm

### Installatie
1. Kloon de repository
2. Installeer afhankelijkheden:
   ```bash
   npm install
   ```
3. Maak een `.env` bestand aan op basis van `.env.example` en voeg je Firebase configuratie toe.

### Ontwikkeling
Start de development server:
```bash
npm run dev
```

### Bouwen voor productie
```bash
npm run build
```

## Beheer
Het beheerpaneel is bereikbaar via `/admin`. Hiervoor moet je ingelogd zijn met een geautoriseerd Google-account.
